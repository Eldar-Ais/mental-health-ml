# Анализ факторов психического здоровья сотрудников IT

Учебный проект по машинному обучению.  
Цель — проанализировать факторы, связанные с обращением сотрудников за лечением психического здоровья, и построить модель классификации уровня риска.

## Описание проекта

В проекте используется открытый датасет [Mental Health in Tech Survey (Kaggle)](https://www.kaggle.com/datasets/osmi/mental-health-in-tech-survey).  
Каждая строка — анонимный ответ сотрудника IT-сферы на анкету о его психическом состоянии и условиях работы.

Основные шаги:

1. Предобработка данных  
   - фильтрация возраста (18–65 лет);  
   - нормализация пола (`Gender_clean`);  
   - преобразование целевой переменной `treatment` из `Yes/No` в `0/1`;  
   - заполнение пропусков и кодирование категориальных признаков.

2. Разведочный анализ данных (EDA)  
   - анализ распределения признаков;  
   - сравнение групп риска по возрасту и семейной истории;  
   - визуализация с помощью гистограмм, boxplot, barplot и ROC-кривых.

3. Модели машинного обучения  
   - Logistic Regression  
   - Decision Tree  
   - SVM (RBF kernel)  

4. Оценка качества  
   - Accuracy, Recall, F1-score;  
   - ROC AUC и ROC-кривые;  
   - анализ важности признаков (Decision Tree).

## Установка и запуск

1. Клонируйте репозиторий или скачайте архив с проектом.
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Скачайте датасет с Kaggle и сохраните файл `survey.csv` в папку `data/`.

4. Запустите проект:

```bash
python src/mental_health_project.py
```

После запуска:
- метрики сохраняются в `model_metrics.csv`;
- графики сохраняются в папку `plots/`.
